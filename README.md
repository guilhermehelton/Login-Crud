<h2>Como iniciar</h2>
<p>Entre na pasta "back-end" e use o comando "npm install", faça o mesmo na pasta "react-crud"</p>
<p>Após isso dentro da pasta "back-end" use "npm run dev" e dentro de "react-crud" use "npm start"</p>
